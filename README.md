# PREVIEW3 - LUMEN Client Preview Site

## Pages
- `index.html` (Manifesto/Home)
- `archive.html` (Work)
- `process.html` (Services)
- `collective.html` (Studio)
- `project-detail.html` (Case Study)

All source pages are in `/site`.

## Local Preview
1. `npm install`
2. `npm run dev`
3. Open the shown local URL.

## Production Build
1. `npm run build`
2. Built static files are generated in `/dist`.
3. Validate locally with `npm run preview`.

## Hosting
- Netlify is preconfigured via `netlify.toml`.
- Any static host can deploy the `/dist` folder.
